## 5. 관찰 사항 (본 WO 범위 외 · 후속 후보)

운영자 콘텐츠 허브 목록의 **"전체 상태" 기본 뷰가 ready 콘텐츠를 포함하지 않는다**(초안만 표시, "완료" 필터로만 ready 확인 가능). 이는 본 WO 변경과 무관한 **기존 목록 조회 동작**이다.

- 영향: 운영자가 draft→ready 전환 시 기본 뷰에서 해당 항목이 사라져 혼동 가능(단, QR·매장 허브 노출 및 "완료" 필터 노출은 정상).
- 권고: 별도 WO로 `/operator/docs` 기본 조회가 draft+ready 를 함께 표시하도록 정합(백엔드 GET `/contents` 기본 status 조건 검토). 본 WO에서는 범위 외로 보고만 함.
