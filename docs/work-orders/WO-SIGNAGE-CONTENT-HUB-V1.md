# Work Order: WO-SIGNAGE-CONTENT-HUB-V1

**Digital Signage 콘텐츠 페이지 정의 및 연결**

---

## 1️⃣ 목적 (Purpose)

현재 Digital Signage 기능은 Core·Extension·API·데이터가 모두 존재함에도 불구하고,
**사용자가 인지할 수 있는 대표 콘텐츠 페이지가 없어 "없는 것처럼 보이는 상태"**이다.

본 Work Order의 목적은:

> **디지털 사이니지용 '콘텐츠 허브 페이지'를 1개 정의하고,
> 모든 대시보드에서 동일한 페이지로 연결하여
> 기능을 "실제로 존재하는 것처럼 보이게" 만드는 것**이다.

---

## 2️⃣ 작업 범위 (Scope)

### ⭕ 포함

* Digital Signage 콘텐츠 허브 **단일 페이지** 정의
* 기존 Global Content API 재사용
* 대시보드 메뉴/라우팅 연결
* UX 명확화 (보기 / 가져오기)

### ❌ 제외

* 새로운 Entity 추가 ❌
* 새로운 API 추가 ❌
* 권한 구조 변경 ❌
* 편집 기능 추가 ❌
* Player / Schedule 수정 ❌

---

## 3️⃣ 신규 정의 페이지

### 📍 페이지 ID (확정)

```
/signage/content
```

> 모든 서비스(Glycopharm / K-Cosmetics / Neture 공통)에서 동일하게 사용

---

## 4️⃣ 페이지 역할 정의 (중요)

### 이 페이지는 무엇인가?

* ❌ 플레이어 페이지 아님
* ❌ 편집 페이지 아님
* ❌ 운영 페이지 아님

### 이 페이지는 이것이다

> **디지털 사이니지용 "동영상 / 동영상 목록을 모아 보고 선택하는 콘텐츠 허브"**

---

## 5️⃣ 페이지 구성 (UX 기준)

### 5.1 상단 탭 구조 (3탭 고정)

```
[ 운영자 콘텐츠 ] [ 공급자 콘텐츠 ] [ 커뮤니티 콘텐츠 ]
```

* 운영자: HQ / 서비스 운영자가 만든 콘텐츠
* 공급자: 네뚜레 공급자 대시보드에서 등록한 콘텐츠
* 커뮤니티: 포럼/공용 등록 콘텐츠

---

### 5.2 각 탭에 표시되는 항목

#### 공통 표시 대상

* 📹 **동영상** (YouTube / Vimeo URL 기반)
* 📂 **동영상 목록 (Playlist)**

#### 카드 정보

* 썸네일 / 미리보기
* 제목
* 제작자
* 출처 (운영자 / 공급자 / 커뮤니티)
* 등록일

#### 주요 액션 버튼

* **[내 대시보드에 가져오기]**

  * Playlist → Playlist Clone
  * Video → Media Clone

---

## 6️⃣ 페이지에서 가능한 행동

| 행동                  | 가능 여부 |
| ------------------- | ----- |
| 콘텐츠 보기              | ⭕     |
| 미리보기                | ⭕     |
| 내 대시보드로 가져오기(Clone) | ⭕     |
| 편집                  | ❌     |
| 삭제                  | ❌     |
| 공용 등록               | ❌     |

> 편집 및 공용 등록은 **각자의 대시보드**에서만 수행

---

## 7️⃣ 대시보드별 연결 규칙

### 7.1 메뉴 연결 (공통)

모든 대시보드에 아래 메뉴 추가 또는 연결:

```
디지털 사이니지 콘텐츠
→ /signage/content
```

### 대상

* 운영자 대시보드
* 공급자 대시보드
* 매장(또는 서비스 운영자) 대시보드

> **모두 동일한 페이지를 본다**

---

## 8️⃣ 기존 기능과의 관계

### 재사용 대상

* Global Content API
* Clone API
* Source (hq / supplier / community)
* Role Guard (읽기 + clone)

### 신규 구현 없음

* API ❌
* Entity ❌
* Role ❌

---

* [ ] "내 대시보드에 가져오기" 정상 동작
* [ ] 모든 대시보드 메뉴에서 동일 페이지로 진입 가능
* [ ] 기존 기능/권한/구조에 영향 없음

---

## 🔚 정리

이 Work Order는 **"무언가를 새로 만드는 작업"이 아니라**,
**이미 완성된 Digital Signage 시스템을 사용자 눈앞에 드러내는 작업**이다.

이 작업이 완료되면:

* "디지털 사이니지가 있는지 모르겠다"는 문제는 즉시 해소된다
* Pilot 협력자에게 바로 안내 가능
* 이후 개선(검색, 추천, 필터 등)도 자연스럽게 확장 가능
