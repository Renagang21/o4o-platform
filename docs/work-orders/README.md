# Work Orders

현재 실행 전·진행 중·반복 참조 가치가 있는 작업요청서 보관 위치입니다.

## 분류 기준

- **이 폴더**: 진행 중 WO, PLAN-ROLE-* 체인, 현재 참조 중인 WO
- **docs/archive/work-orders/**: 완료된 일회성 WO 이력
- **docs/archive/obsolete/work-orders/**: 폐기된 정책·서비스 전제 WO

## 유지 대상

- `PLAN-ROLE-*`: Phase 3~5 RBAC 정비 설계 (실행 전)
- `RBAC-DATA-NORMALIZATION-*`: RBAC 정규화 기록
- `WO-CHANNEL-*`: CLAUDE.md 직접 링크 (이동 금지)
- 판단 보류 문서: 상태 불명확, 현재 위치 유지 중
