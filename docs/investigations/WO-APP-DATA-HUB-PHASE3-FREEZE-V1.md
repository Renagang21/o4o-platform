# WO-APP-DATA-HUB-PHASE3-FREEZE-V1

> **작업일**: 2026-02-11
> **성격**: Phase 3 기준선 고정 선언
> **상태**: Phase 3 기준선 고정 완료

---

## 0. 목적 (Purpose)

APP Data Hub의 **Phase 3 (3-A + 3-B)** 결과를
하나의 **완결된 AI/자동화 기준선**으로 고정한다.

Phase 2에서 확정된 "사람이 선택하는 복사 UX" 위에
**"AI가 제안하되, 선택은 사람이 하는"** 패턴을 완성하여 고정한다.

본 문서 이후의 변경은 **Phase 4** 또는 별도 확장 단계로만 진행한다.

---

## 1. Phase 3 기준선 선언 (Declaration)

**APP Data Hub – Phase 3는 다음 상태로 고정된다.**

* AI 추천 배너 (Non-blocking advisor) 완성
* 추천 근거 토글 표시 완성
* 조건부 자동 기본값 적용 완성
* 사용자 수동 변경 시 auto-default 즉시 해제 완성
* Quick Actions 행동 기반 자동 정렬 완성

👉 이 시점 이후:

* AI 추천 로직 구조 변경 ❌
* 자동 기본값 적용 조건 변경 ❌
* 추천 UI 패턴 변경 ❌
  (단, 버그 수정은 허용)

## 4. UX 기준선 (고정)

### 4-1. AI 추천 원칙

* ❌ "AI가 정해줍니다" 느낌 금지
* ❌ 추천 숨김 금지
* ✅ "도움 제안" 톤 유지
* ✅ 사용자가 바꾸면 즉시 추천 상태 해제

### 4-2. 자동 기본값 원칙

* ✅ 보수적 기준만 자동 적용 (LIMITED, 승인 완료 REQUEST)
* ✅ 모달 최초 진입 시 1회만
* ✅ 사용자 수동 변경 즉시 해제
* ❌ OPEN / DISPLAY_ONLY는 추천만 (자동 기본값 아님)

| 항목 | 기준 |

### 고정된 파일

| 파일 | 역할 |
|------|------|
| `store-ai-recommend.ts` | 복사 옵션 추천 + 자동 적용 판정 |
| `store-ai-summary.ts` | 매장 요약 (우선순위 문장 + 행동 가중치) |
| `store-action-sort.ts` | Quick Actions 자동 정렬 + 가중치 계산 |
| `HubCopyModal.tsx` | AI 추천 배너 + 자동 기본값 + 근거 표시 |

## 9. 패턴 재사용 선언

Phase 3에서 확립된 다음 패턴은
**다른 서비스에서도 동일하게 적용 가능**한 재사용 가능 패턴이다:

| 패턴 | 설명 |
|------|------|
| Non-blocking Advisor | AI는 제안만, 적용은 사용자 |
| Conditional Auto-default | 보수적 조건에서만 기본값 자동 적용 |
| Action-weighted Sort | 행동 로그 기반 UI 요소 자동 정렬 |
| Reason Toggle | 추천 근거 투명 공개 |
