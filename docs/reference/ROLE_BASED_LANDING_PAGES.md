## 1️⃣ Customer (일반 사용자/구매자)

### 📍 로그인 후 이동 경로
```
/my-account (또는 /dashboard/customer)
```

### 🎨 페이지 구성 요소

#### **상단 요약 섹션**
```
┌─────────────────────────────────────────┐
│  안녕하세요, [사용자명]님! 👋             │
│  포인트: 5,000P | 쿠폰: 3개              │
└─────────────────────────────────────────┘
```

#### **메인 대시보드 카드 (Grid Layout)**

| 카드 | 내용 | 링크 | 아이콘 |
|------|------|------|--------|
| **주문 내역** | 최근 주문 1-3개 미리보기<br/>- 주문번호<br/>- 상품명<br/>- 배송 상태<br/>"모두 보기" 버튼 | `/my-account/orders` | 📦 |
| **배송 추적** | 배송 중인 상품 실시간 추적<br/>- 택배사 정보<br/>- 현재 위치<br/>- 예상 도착일 | `/my-account/tracking` | 🚚 |
| **위시리스트** | 저장한 상품 수<br/>- 가격 하락 알림<br/>- 재입고 알림<br/>최근 본 상품 | `/my-account/wishlist` | ❤️ |
| **포인트/쿠폰** | 사용 가능한 포인트<br/>사용 가능한 쿠폰 목록<br/>쿠폰 다운로드 센터 | `/my-account/rewards` | 🎁 |
| **1:1 문의** | 미답변 문의 개수<br/>최근 문의 내역<br/>"새 문의하기" 버튼 | `/my-account/support` | 💬 |
| **리뷰 작성** | 리뷰 작성 대기 상품<br/>작성한 리뷰 통계<br/>리뷰 포인트 적립 안내 | `/my-account/reviews` | ⭐ |

#### **개인화 추천 섹션**
- **다시 구매**: AI 기반 재구매 추천 (Amazon 방식)
  - "다시 살 시간이에요!" 섹션
  - 이전 구매 상품 빠른 재주문 버튼

- **맞춤 추천**:
  - 최근 본 상품과 유사한 상품
  - 관심 카테고리 신상품

#### **Quick Actions (빠른 작업)**
- 🔍 주문 조회 (주문번호 입력)
- 📝 취소/반품 신청
- 💳 결제 수단 관리
- 📍 배송지 관리

### 📊 통계/분석 섹션
```
┌─────────────────────────┐
│ 올해 구매 통계          │
```
