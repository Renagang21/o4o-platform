# platform/

## 디렉토리 목적

**기능 및 서비스별 상세 스펙 및 실행 문서**

이 디렉토리는 o4o-platform의 특정 기능과 서비스에 대한 설계 사양서, UX 플로우, 실행 계획 등을 포함합니다.

## 주요 내용

- **Digital Signage**: 디지털 사이니지 기능 스펙, 실행 계획, UX 플로우
- **LMS**: 학습 관리 시스템 연결 기준
- **Forum**: 포럼 연결 기준
- **기타 서비스별 스펙 문서**

## 관련 디렉토리

- `platform-core/`: 플랫폼 전반의 기준 문서, 아키텍처, 정책
- `services/`: 구현된 서비스별 상세 문서
- `architecture/`: 전체 시스템 아키텍처

## 문서 규칙

- 버전 관리: `v1`, `v0.1` 등 명확한 버전 표기
- 파일명: `kebab-case.md` 형식
- 기능별 하위 디렉토리 활용
