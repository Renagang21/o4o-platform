## 1. 개요

이 문서는 **약국(의료기관)** 사용자를 위한 Digital Signage 운영 가이드입니다.
약국은 콘텐츠의 **선택·배치 주체**로서, HQ에서 제공하는 콘텐츠 중 원하는 것을 선택하여 자율적으로 디스플레이를 운영합니다.

### 핵심 원칙

> **"약국이 선택하고, 약국이 배치한다"**

- 본부(HQ)는 콘텐츠를 **제공**합니다
- 약국은 제공된 콘텐츠 중 원하는 것을 **선택**합니다
- 약국이 직접 디스플레이 구성을 **배치**합니다
- **약국이 재생 결과에 대한 1차 책임을 집니다** (강제 콘텐츠 제외)

## 2. 콘텐츠 흐름: Clone → Select → Arrange → Play

```
┌────────────┐                ┌────────────┐                ┌────────────┐
│  글로벌    │    Clone      │   내 약국   │    Select     │   활성     │
│  콘텐츠    │ ───────────>  │  라이브러리 │ ───────────>  │  콘텐츠    │
│  (HQ/공급) │               │   (복사본)  │               │   (사용중) │
└────────────┘               └────────────┘               └────────────┘
                                                                 │
                                                                 │ Arrange
                                                                 ▼
                                                          ┌────────────┐
                                                          │  디스플레이 │
                                                          │   (재생)   │
                                                          └────────────┘
```

### 2.1 Clone (복제) - "내 라이브러리로 가져오기"

글로벌 콘텐츠 브라우저에서 마음에 드는 콘텐츠를 내 약국 라이브러리로 복사합니다.

**방법**:
1. `/signage/store/global` 페이지 접속
2. HQ / Supplier / Community 탭에서 콘텐츠 탐색
3. 원하는 콘텐츠의 **"내 Signage에 추가"** 버튼 클릭
4. 내 라이브러리에 복사본 생성됨

### 2.2 Select (선택)

복제한 콘텐츠 중 실제로 사용할 것을 선택합니다.

**방법**:
1. `/signage/store/playlists` 페이지 접속
2. 새 플레이리스트 생성 또는 기존 플레이리스트 편집
3. 라이브러리에서 콘텐츠 선택하여 추가
4. 저장

### 2.3 Arrange (배치) - "채널에 배치"

선택한 콘텐츠의 순서와 스케줄을 설정합니다.

**방법**:
1. 플레이리스트 내 항목 순서 드래그앤드롭
2. `/signage/store/schedules`에서 스케줄 생성
3. 시간대별 플레이리스트 지정
4. **채널에 배치**하여 디스플레이에 연결

### 2.4 Play (재생)

설정한 대로 디스플레이에서 재생됩니다.

## 3. UI 용어 안내

### 3.1 버튼 문구

| 버튼 | 의미 |
|------|------|
| **내 Signage에 추가** | 글로벌 콘텐츠를 내 라이브러리로 복제 |
| **내 라이브러리로 가져오기** | 콘텐츠 복제 동작 |
| **채널에 배치** | 편성을 특정 디스플레이에 연결 |
| **필수 노출 콘텐츠** | 제거 불가한 강제 광고 |

### 3.2 상태 문구

| 상태 | 의미 |
|------|------|
| **현재 방영 중** | 지금 디스플레이에서 재생되고 있음 |
| **편성됨 (비활성)** | 목록에는 있지만 재생 안 됨 |
| **필수 노출 (제거 불가)** | 강제 콘텐츠, 삭제/비활성화 불가 |
| **본부 제공 콘텐츠** | HQ에서 제작한 글로벌 콘텐츠 |
| **공급자 제공 콘텐츠** | 공급업체에서 제공한 콘텐츠 |
| **다른 약국 공유** | Community에서 공유된 콘텐츠 |

## 4. 주요 기능

### 4.1 내 플레이리스트 관리

| 기능 | 설명 | 경로 |
|------|------|------|
| 플레이리스트 목록 | 내 플레이리스트 조회 | `/signage/store/playlists` |
| 플레이리스트 생성 | 새 플레이리스트 만들기 | `/signage/store/playlists/new` |
| 플레이리스트 편집 | 기존 플레이리스트 수정 | `/signage/store/playlists/:id` |
| 플레이리스트 삭제 | 불필요한 플레이리스트 제거 | 목록에서 삭제 |

### 4.2 글로벌 콘텐츠 탐색

| 소스 | 설명 | 탭 |
|------|------|-----|
| HQ | 본부 제공 콘텐츠 (공식) | HQ 탭 |
| Supplier | 공급자 제공 콘텐츠 | Supplier 탭 |
| Community | 다른 약국 공유 콘텐츠 | Community 탭 |

### 4.3 스케줄 관리

| 기능 | 설명 |
|------|------|
| 시간대 설정 | 아침/점심/저녁 등 시간대별 다른 콘텐츠 |
| 요일 설정 | 평일/주말 다른 콘텐츠 |
| 특별 이벤트 | 특정 날짜 이벤트 스케줄 |

### 4.4 디바이스 관리

| 기능 | 설명 |
|------|------|
| 디스플레이 등록 | 약국 내 디스플레이 장치 등록 |
| 채널 설정 | 디스플레이 슬롯에 채널 연결 |
| 상태 모니터링 | 장치 온라인/오프라인 상태 확인 |

## 5. 필수 노출 콘텐츠 (Forced Items)

본부에서 지정한 **필수 노출 콘텐츠**는 `isForced` 플래그가 설정되어 있습니다.

### 5.1 특징

| 특성 | 설명 |
|------|------|
| 삭제 불가 | 플레이리스트에서 제거할 수 없음 |
| 수정 불가 | 콘텐츠 내용 변경 불가 |
| 순서 변경 | 필수 콘텐츠끼리의 순서만 변경 가능 |
| 비활성화 불가 | 항상 재생됨 |

### 5.2 표시 방법

필수 노출 콘텐츠는 플레이리스트에서 특별하게 표시됩니다:
- 자물쇠 아이콘 (🔒)
- **"필수 노출 (제거 불가)"** 라벨
- 배경색 구분 (주황색/빨간색 계열)

### 5.3 책임 귀속

| 콘텐츠 유형 | 책임 주체 |
|------------|----------|
| 필수 노출 콘텐츠 | 본부/운영자 |
| 일반 콘텐츠 (약국 선택) | **약국** |

> 필수 노출 콘텐츠에 대한 문의는 본부/운영자에게 전달됩니다.

---

## 6. 책임 안내

### 6.1 약국의 책임

약국은 다음에 대해 책임을 집니다:
- 직접 선택한 콘텐츠의 적절성
- 편성 설정으로 인한 재생 결과
- 직접 등록한 콘텐츠의 저작권 적법성

### 6.2 약국이 책임지지 않는 것

- 필수 노출 콘텐츠(isForced)의 내용
- 시스템 오류로 인한 재생 문제
- HQ/공급자 제공 콘텐츠의 저작권 문제

### 6.3 고지 사항

> **콘텐츠 편성 안내**
>
> 본 Signage에 표시되는 콘텐츠는 귀하가 선택·편성한 결과입니다.
> 필수 노출 콘텐츠를 제외한 모든 콘텐츠의 재생 결과에 대한
> 책임은 약국에 있습니다.

## 7. 자주 묻는 질문 (FAQ)

### Q1: 본부에서 콘텐츠를 직접 설정해주나요?

**아니요.** 본부는 콘텐츠를 **제공**만 합니다. 약국이 직접 선택하고 배치합니다.

### Q2: 원하지 않는 콘텐츠가 나오면 어떻게 하나요?

필수 노출 콘텐츠가 아니라면, 플레이리스트에서 제거하면 됩니다.
필수 노출 콘텐츠는 본부 정책에 따라 반드시 재생되며, 문의는 본부로 전달됩니다.

### Q3: 내가 만든 콘텐츠를 다른 약국과 공유할 수 있나요?

네. Community에 제출하면 승인 후 다른 약국이 복제할 수 있습니다.
공유한 콘텐츠의 저작권 적법성은 공유한 약국이 책임집니다.

### Q4: 글로벌 콘텐츠를 수정할 수 있나요?

원본은 수정할 수 없지만, **"내 라이브러리로 가져오기"** 후 복사본을 자유롭게 수정할 수 있습니다.

### Q5: 디스플레이가 여러 대인 경우 어떻게 하나요?

각 디스플레이에 다른 채널을 연결하여 다른 콘텐츠를 재생할 수 있습니다.

### Q6: 재생 결과에 문제가 있으면 누구에게 문의하나요?

| 문제 유형 | 문의처 |
|----------|--------|
| 필수 노출 콘텐츠 관련 | 본부/운영자 |
| 시스템/장치 오류 | 플랫폼 고객센터 |
| 일반 콘텐츠 편성 | 약국 자체 관리 |
