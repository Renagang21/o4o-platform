# Signage Metrics Responsibility V1

> Version: 1.0
> Date: 2026-01-19
> Status: Active
> Reference: ROLE-DEFINITION-V1.2

---

## 1. 문서 목적

이 문서는 Digital Signage **성과 지표의 제공 범위와 책임 경계**를 정의합니다.

Signage 지표는 **측정 가능해지는 순간 책임이 발생**합니다.
따라서 **지표는 제공하되, 성과 책임은 분리**해야 합니다.

---

## 2. 핵심 원칙

### 2.1 기본 선언

> **Signage 성과 지표는
> 시스템 재생 로그를 기반으로 제공되며,
> 실제 시청 효과나 매출 성과를 보장하지 않습니다.**

### 2.2 지표의 성격

플랫폼이 제공하는 모든 Signage 지표는:
- **시스템 로그 기반 참고 지표**
- **실제 효과 보증이 아님**
- **해석 및 활용 책임은 사용자에게 있음**

---

## 3. 플랫폼이 제공하는 지표

### 3.1 제공 지표 목록

| 지표 | 설명 | 계산 방식 |
|------|------|----------|
| 재생 시도 횟수 | 콘텐츠 재생 시도 로그 수 | API 호출 수 |
| 플레이리스트 회전 수 | 전체 플레이리스트 1회전 횟수 | 루프 완료 카운트 |
| 콘텐츠 노출 시간 | 각 콘텐츠의 총 재생 시간(계산값) | 재생 횟수 × 콘텐츠 길이 |
| 채널별 재생 비율 | 채널별 콘텐츠 재생 분포 | 채널별 로그 집계 |
| 콘텐츠별 재생 비율 | 개별 콘텐츠 재생 점유율 | 콘텐츠별 로그 집계 |
| 시간대별 재생 현황 | 시간대별 재생 분포 | 시간대별 로그 집계 |

### 3.2 지표의 한계

| 지표 | 측정 가능 | 한계 |
|------|----------|------|
| 재생 시도 | O | 실제 화면 출력 여부 불확실 |
| 재생 완료 | O (추정) | 중간 이탈 감지 한계 |
| 시청 여부 | X | TV 앞 실제 시청자 불명 |
| 주목도 | X | 시청자 집중도 측정 불가 |
| 광고 효과 | X | 행동 전환 측정 불가 |

---

## 4. 플랫폼이 책임지지 않는 것

### 4.1 명시적 면책 항목

플랫폼은 다음에 대해 **책임지지 않습니다**:

| 항목 | 이유 |
|------|------|
| 실제 시청 여부 | TV 앞 시청자 유무 측정 불가 |
| 고객 주목도 | 시청자 집중도 측정 불가 |
| 매출 전환 | 시청과 구매 간 인과관계 증명 불가 |
| 광고 효과 | 브랜드 인지/행동 변화 측정 불가 |
| KPI 달성 | 외부 요인 통제 불가 |

### 4.2 면책 선언

> 플랫폼이 제공하는 지표는 시스템 로그 기반 참고 자료입니다.
> 지표를 기반으로 한 비즈니스 의사결정 및 그 결과에 대해
> 플랫폼은 책임을 지지 않습니다.

## 5. 책임 분기 구조

### 5.1 책임 분기표

| 항목 | 책임 주체 |
|------|----------|
| 지표 계산 로직 정확성 | **플랫폼** |
| 지표 데이터 무결성 | **플랫폼** |
| 지표 해석 | **약국 / 운영자** |
| 지표 기반 의사결정 | **약국 / 운영자** |
| 성과 결과 | **약국** |
| 강제 광고 KPI | **운영자/HQ** |

### 5.2 책임 영역 다이어그램

```

```

## 6. 역할별 지표 활용

### 6.1 약국(Store)

| 활용 가능 지표 | 활용 목적 | 주의사항 |
|--------------|----------|----------|
| 재생 현황 | 편성 확인 | 참고용 |
| 채널별 비율 | 채널 밸런스 확인 | 참고용 |
| 시간대별 현황 | 스케줄 최적화 | 참고용 |

### 6.2 운영자(HQ)

| 활용 가능 지표 | 활용 목적 | 주의사항 |
|--------------|----------|----------|
| 글로벌 콘텐츠 복제율 | 콘텐츠 인기도 | 참고용 |
| 강제 광고 도달율 | 필수 콘텐츠 커버리지 | HQ 책임 |
| 전체 재생 통계 | 서비스 현황 | 집계값만 |

### 6.3 공급자(Supplier)

| 활용 가능 지표 | 활용 목적 | 주의사항 |
|--------------|----------|----------|
| 공급 콘텐츠 복제율 | 콘텐츠 활용도 | 참고용 |
| 공급 콘텐츠 재생 추정 | 노출 추정 | 보장 아님 |

## 7. 강제 광고 KPI 특례

### 7.1 강제 광고 지표 책임

`isForced = true` 콘텐츠에 대해서는:

| 지표 | 책임 |
|------|------|
| 노출 기회 제공 | 플랫폼 |
| 도달율 목표 | 운영자/HQ |
| 실제 시청 효과 | 면책 (측정 불가) |

### 7.2 강제 광고 보고

운영자/HQ에게 제공되는 강제 광고 지표:
- 강제 콘텐츠 포함 플레이리스트 수
- 강제 콘텐츠 재생 시도 횟수(추정)
- 매장 도달율(복제 기준)

---

## 8. 약관 반영 문구

### 8.1 서비스 이용약관 반영

> **제X조 (Signage 성과 지표)**
>
> 1. 플랫폼이 제공하는 Signage 성과 지표는 시스템 재생 로그를 기반으로 산출된 참고 자료입니다.
>
> 2. 지표는 실제 시청 여부, 광고 효과, 매출 전환 등을 보장하지 않습니다.
>
> 3. 지표의 해석 및 이를 기반으로 한 비즈니스 의사결정에 대한 책임은 사용자에게 있습니다.
>
> 4. 지표 데이터의 계산 로직 및 무결성에 대해서만 플랫폼이 책임집니다.

### 8.2 면책 조항

> 플랫폼은 다음에 대해 책임을 지지 않습니다:
> - 지표 기반 비즈니스 의사결정의 결과
> - 지표와 실제 비즈니스 성과 간의 괴리
> - 지표 해석 오류로 인한 손실
> - 외부 요인으로 인한 지표 변동

## 9. 고지 문구

### 9.1 Analytics 대시보드 고지

Analytics 화면에 표시:

> **지표 안내**
>
> 표시되는 지표는 시스템 로그 기반 참고 자료입니다.
> 실제 시청 효과나 매출 성과를 보장하지 않습니다.

### 9.2 리포트 다운로드 시 고지

리포트 파일에 포함:

> **면책 고지**
>
> 본 리포트의 데이터는 시스템 로그 기반 산출값입니다.
> 실제 시청, 광고 효과, 매출 전환을 보장하는 수치가 아닙니다.
> 참고 자료로만 활용해 주세요.
