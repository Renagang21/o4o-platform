# Community Content Rules V1

> Version: 1.0
> Date: 2026-01-19
> Status: Active
> Reference: ROLE-DEFINITION-V1.2

---

## 1. 개요

이 문서는 Digital Signage **Community 콘텐츠 공유** 기능의 규칙을 정의합니다.
Community는 매장 간 콘텐츠를 **직접 배포하는 기능이 아니라**,
**공유 가능한 콘텐츠를 글로벌 라이브러리에 노출하는 구조**입니다.

---

## 2. Community의 정의

### 2.1 목적

- 매장이 자체 제작한 유용한 콘텐츠를 다른 매장과 공유
- 우수 콘텐츠의 플랫폼 내 확산 촉진
- 매장 간 콘텐츠 아이디어 교류

### 2.2 핵심 원칙

> **Community는 배포가 아닌 공유입니다.**

- 원본 매장이 **공유를 선택**
- HQ가 **승인/큐레이션**
- 다른 매장이 **선택적으로 복제**
- **강제 배포 없음**

---

## 3. 공유 흐름

### 3.1 전체 흐름

```
┌──────────┐     Submit      ┌──────────────┐     Review     ┌──────────────┐
│ Store A  │ ─────────────> │  Community   │ ─────────────> │     HQ       │
│ (Creator)│                │   Queue      │                │   Review     │
└──────────┘                └──────────────┘                └──────────────┘
                                                                   │
                                                                   │ Approve
                                                                   ▼
┌──────────┐     Clone      ┌──────────────┐                ┌──────────────┐
│ Store B  │ <───────────── │   Global     │ <───────────── │  Published   │
│ (User)   │                │   Library    │                │  Community   │
└──────────┘                └──────────────┘                └──────────────┘
```

### 3.2 단계별 설명

| 단계 | 주체 | 설명 |
|------|------|------|
| 1. Submit | Store A | 자체 콘텐츠를 Community 공유 대상으로 등록 |
| 2. Queue | System | 승인 대기열에 추가 |
| 3. Review | HQ | 품질/적합성/저작권 검토 |
| 4. Approve/Reject | HQ | 승인 또는 사유와 함께 반려 |
| 5. Publish | System | 글로벌 라이브러리 Community 탭에 공개 |
| 6. Clone | Store B | 원하는 콘텐츠를 자신의 라이브러리로 복제 |

## 4. 규칙

### 4.1 공유 규칙

| 규칙 | 설명 |
|------|------|
| **소유권 유지** | 원 제작 매장이 소유권 유지 |
| **수정 권한** | 원 제작 매장만 원본 수정/삭제 가능 |
| **복제 후 독립** | 복제 후 각 매장이 독립적으로 관리 |
| **책임 분리** | 복제 후 편성 결과는 복제한 매장 책임 |

### 4.2 복제 규칙

| 규칙 | 설명 |
|------|------|
| **선택적 복제** | 매장이 원할 때만 복제 |
| **자동 배포 없음** | Community 콘텐츠는 자동으로 매장에 추가되지 않음 |
| **수정 자유** | 복제 후 자유롭게 수정 가능 |
| **원본 연결** | parentId로 원본 추적 가능 (선택사항) |

### 4.3 승인 규칙

| 기준 | 설명 |
|------|------|
| **품질** | 해상도, 가독성, 전문성 |
| **적합성** | 브랜드 가이드라인 준수 |
| **내용** | 부적절한 내용 없음 |
| **저작권** | 저작권 문제 없음 |

---

## 5. 금지 사항

### 5.1 절대 금지

| 금지 사항 | 이유 |
|----------|------|
| Community 콘텐츠에 `isForced` 설정 | 강제 배포 구조 금지 |
| 특정 매장에 자동 배포 | 선택적 복제 원칙 |
| 매장 동의 없이 적용 | 자율성 원칙 |
| 원본 제작자 외 수정 | 소유권 원칙 |

### 5.2 시스템 차단

다음 동작은 시스템에서 차단됩니다:

```typescript
// Community 콘텐츠는 isForced 설정 불가
if (content.source === 'community' && payload.isForced === true) {
  throw new Error('Community content cannot be forced');
}

// Community 콘텐츠는 자동 배포 불가
if (content.source === 'community' && action === 'auto-deploy') {
  throw new Error('Community content cannot be auto-deployed');
}
```

## 6. 책임 구조

### 6.1 공유자 (원 제작 매장)

공유자는 다음에 대해 책임집니다:
- 공유 콘텐츠의 저작권 적법성
- 공유 콘텐츠의 내용 적절성
- 부적절한 콘텐츠 공유에 대한 책임

### 6.2 복제자 (다른 매장)

복제자는 다음에 대해 책임집니다:
- 복제 후 자신의 디스플레이 재생 결과
- 복제 콘텐츠 수정 후의 내용
- 자체 편성 결과

### 6.3 HQ/운영자

HQ는 다음에 대해 책임집니다:
- 승인 과정에서의 검토 품질
- 명백히 부적절한 콘텐츠 승인 시 공동 책임

### 6.4 책임 매트릭스

| 상황 | 책임 주체 |
|------|----------|
| 공유 콘텐츠 저작권 위반 | 공유자 (원 제작 매장) |
| 복제 후 편성 결과 | 복제자 (복제한 매장) |
| 승인된 부적절 콘텐츠 | 공유자 + HQ (공동) |
| 복제 후 수정된 콘텐츠 | 복제자 (수정한 매장) |

## 7. 약관 반영 사항

### 7.1 공유 약관

> **커뮤니티 공유 약관**
>
> 1. 커뮤니티에 콘텐츠를 공유하는 매장은 해당 콘텐츠의 저작권 적법성을 보증합니다.
>
> 2. 공유된 콘텐츠로 인해 발생하는 법적 문제의 1차 책임은 공유한 매장에 있습니다.
>
> 3. 부적절한 콘텐츠 공유 시 공유 권한이 제한될 수 있습니다.

### 7.2 복제 약관

> **커뮤니티 복제 약관**
>
> 1. 커뮤니티 콘텐츠를 복제하는 매장은 복제 이후 자신의 디스플레이 재생 결과에 책임을 집니다.
>
> 2. 복제된 콘텐츠는 원본 제작자와 무관하게 독립적으로 관리됩니다.
>
> 3. 복제 후 수정된 내용에 대한 책임은 수정한 매장에 있습니다.

## 8. UI 표시

### 8.1 Community 탭 표시

글로벌 콘텐츠 브라우저에서 Community 탭:
- 소스 라벨: **"다른 약국 공유"**
- 원 제작자 표시 (익명화 옵션 가능)
- 복제 수 표시

### 8.2 복제 시 안내

복제 버튼 클릭 시 안내:
> 이 콘텐츠를 **내 라이브러리로 가져오기**하시겠습니까?
> 복제 후 편성 결과에 대한 책임은 귀하에게 있습니다.

### 8.3 공유 시 안내

공유 제출 시 안내:
> 이 콘텐츠를 **커뮤니티에 공유**하시겠습니까?
> 공유된 콘텐츠의 저작권 적법성에 대한 책임은 귀하에게 있습니다.

## 9. 관리자 기능

### 9.1 HQ 승인 관리

**경로**: `/signage/hq/community`

| 기능 | 설명 |
|------|------|
| 대기열 조회 | 승인 대기 콘텐츠 목록 |
| 상세 검토 | 콘텐츠 미리보기 및 정보 확인 |
| 승인 | 글로벌 라이브러리에 공개 |
| 거부 | 사유와 함께 반려 |
| 신고 처리 | 신고된 콘텐츠 검토 및 조치 |

### 9.2 신고 처리

매장이 Community 콘텐츠를 신고할 수 있습니다:
- 저작권 문제
- 부적절한 내용
- 품질 문제

신고 시 HQ가 검토 후 조치합니다.
