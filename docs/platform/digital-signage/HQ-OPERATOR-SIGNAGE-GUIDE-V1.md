## 1. 개요

이 문서는 **본부(HQ) 운영자**를 위한 Digital Signage 운영 가이드입니다.
HQ 운영자는 글로벌 콘텐츠의 **생산 주체**로서, 전국 매장이 활용할 콘텐츠를 제작하고 관리합니다.

### 핵심 원칙

> **"본부는 만들고, 매장이 선택한다"**

- HQ는 글로벌 콘텐츠를 **생산**합니다
- 매장은 HQ 콘텐츠를 **복제**하여 **선택적으로** 사용합니다
- HQ의 매장 통제는 **필수 노출 콘텐츠(isForced)** 로만 제한됩니다
- HQ는 **필수 노출 콘텐츠에 대해서만** 책임을 집니다

## 2. HQ의 역할과 책임

### 2.1 역할 범위

| 영역 | HQ 역할 | 책임 |
|------|---------|------|
| 콘텐츠 생성 | **담당** | 콘텐츠 품질 및 적법성 |
| 콘텐츠 선택 | 해당 없음 | 매장 책임 |
| 콘텐츠 배치 | 해당 없음 | 매장 책임 |
| 필수 노출 콘텐츠 | **담당** | **전적으로 HQ 책임** |

### 2.2 명확한 경계

| 허용 | 금지 |
|------|------|
| 글로벌 콘텐츠 CRUD | 개별 매장 데이터 접근 |
| 필수 노출 콘텐츠 지정 | 매장 플레이리스트 편집 |
| Community 콘텐츠 승인 | 매장 스케줄 관리 |
| HQ 분석 조회 | 매장 디바이스 설정 |

### 2.3 책임 구조

| 콘텐츠 유형 | 책임 주체 |
|------------|----------|
| 필수 노출 콘텐츠 (isForced=true) | **HQ/운영자** |
| 일반 글로벌 콘텐츠 (매장이 선택한 경우) | 매장 |
| 매장 자체 등록 콘텐츠 | 매장 |

## 3. UI 용어 표준

### 3.1 사용해야 할 문구

| 상황 | 올바른 문구 | 영문 |
|------|------------|------|
| 콘텐츠 공개 | **글로벌 라이브러리에 추가** | Add to Global Library |
| 콘텐츠 제공 | **복제 가능 콘텐츠로 제공** | Provide as Clonable Content |
| 필수 지정 | **필수 노출로 지정** | Mark as Required |

### 3.2 금지된 문구

다음 문구는 잘못된 책임 구조를 암시하므로 **사용 금지**:

| 금지 문구 | 이유 |
|----------|------|
| "매장 적용" | HQ가 매장을 대신 편성하는 것처럼 오해 |
| "매장에 배포" | 강제 배포 구조로 오해 |
| "매장 관리" | HQ가 매장 콘텐츠를 관리하는 것처럼 오해 |
| "대신 운영" | 자율성 원칙 위반 |

## 4. 콘텐츠 생산 워크플로우

### 4.1 플레이리스트 생성

```
계획 → 제작 → 검토 → 글로벌 라이브러리에 추가 → 모니터링
```

**단계별 가이드**:

1. **계획**: 캠페인/프로모션 기획
2. **제작**: `/signage/hq/playlists/new`에서 플레이리스트 생성
3. **검토**: 내부 검토 및 승인
4. **글로벌 라이브러리에 추가**: 매장이 복제할 수 있도록 공개
5. **모니터링**: 매장 복제/사용 현황 확인

### 4.2 미디어 관리

| 기능 | 경로 | 설명 |
|------|------|------|
| 업로드 | `/signage/hq/media/upload` | 새 미디어 업로드 |
| 라이브러리 | `/signage/hq/media` | 미디어 관리 |
| 상세 | `/signage/hq/media/:id` | 개별 미디어 편집 |

### 4.3 템플릿 관리

| 기능 | 경로 | 설명 |
|------|------|------|
| 목록 | `/signage/hq/templates` | 템플릿 목록 |
| 생성 | `/signage/hq/templates/new` | 새 템플릿 |
| 편집 | `/signage/hq/templates/:id` | 템플릿 수정 |

## 5. 필수 노출 콘텐츠 (Forced Items) 관리

### 5.1 정의

`isForced: true` 플래그가 설정된 콘텐츠는 매장이 **삭제/수정/비활성화할 수 없습니다**.
이는 HQ가 전 매장에 **필수 콘텐츠**를 전달하는 유일한 통제 수단입니다.

### 5.2 책임

> **필수 노출 콘텐츠에 대한 모든 책임은 HQ/운영자에게 있습니다.**

- 콘텐츠 내용의 적법성
- 광고법 준수
- 저작권 문제
- 매장 문의 대응

### 5.3 설정 방법

**경로**: `/signage/hq/forced-items`

1. Forced Items 관리 페이지 접속
2. **필수 노출로 지정**할 플레이리스트 항목 선택
3. `isForced` 플래그 활성화
4. 적용 범위 설정 (전체/특정 서비스)
5. 저장

### 5.4 사용 원칙

| 원칙 | 설명 |
|------|------|
| 최소한의 사용 | 꼭 필요한 경우에만 지정 |
| 명확한 목적 | 법적 의무, 안전 정보, 핵심 프로모션 |
| 기간 제한 | 불필요하게 오래 유지하지 않음 |
| 투명성 | 매장에 필수 노출 사유 안내 |

### 5.5 매장에서의 표시

매장 플레이리스트에서:
- 삭제 버튼 비활성화
- 편집 버튼 비활성화
- 자물쇠 아이콘 표시
- **"필수 노출 (제거 불가)"** 라벨 표시

## 6. Community 콘텐츠 승인

### 6.1 정의

Community는 **매장 간 콘텐츠를 직접 배포하는 기능이 아닙니다**.
공유 가능한 콘텐츠를 글로벌 라이브러리에 노출하는 구조입니다.

### 6.2 승인 워크플로우

```
매장 제출 → 대기열 → HQ 검토 → 승인/거부 → 글로벌 라이브러리 공개
```

**경로**: `/signage/hq/community`

### 6.3 검토 기준

| 기준 | 설명 |
|------|------|
| 품질 | 해상도, 가독성, 전문성 |
| 적합성 | 브랜드 가이드라인 준수 |
| 내용 | 부적절한 내용 없음 |
| 권리 | 저작권 문제 없음 |

### 6.4 금지 사항

- Community 콘텐츠에 `isForced` 설정 **금지**
- 특정 매장에 자동 배포 **금지**
- 매장 동의 없이 적용 **금지**

## 7. 분석 및 모니터링

### 7.1 HQ 분석 대시보드

**경로**: `/signage/hq/analytics`

| 지표 | 설명 |
|------|------|
| 복제 현황 | 글로벌 콘텐츠별 복제 수 |
| 활용률 | 복제 후 실제 사용률 |
| 인기 콘텐츠 | 가장 많이 복제된 콘텐츠 |
| 매장 커버리지 | 콘텐츠 도달 매장 비율 |

### 7.2 유의사항

- HQ는 **집계된 통계**만 조회 가능
- 개별 매장의 상세 데이터에는 접근 불가
- 매장 프라이버시 보호

## 9. 자주 묻는 질문 (FAQ)

### Q1: 매장 디스플레이에 직접 콘텐츠를 설정할 수 있나요?

**아니요.** HQ는 글로벌 콘텐츠를 **복제 가능 콘텐츠로 제공**하고, 매장이 직접 선택·배치합니다.
꼭 나가야 하는 콘텐츠는 **필수 노출로 지정**하세요.

### Q2: 특정 매장에만 콘텐츠를 전달하고 싶어요.

필수 노출 콘텐츠의 적용 범위를 설정할 때 특정 지역/그룹을 지정할 수 있습니다.
단, 일반 콘텐츠는 모든 매장이 자유롭게 선택할 수 있습니다.

### Q3: 매장이 콘텐츠를 잘 사용하고 있는지 어떻게 확인하나요?

HQ Analytics에서 복제 현황과 활용률을 확인할 수 있습니다.
개별 매장 데이터는 프라이버시 보호를 위해 접근이 제한됩니다.

### Q4: 매장에서 부적절한 콘텐츠를 만들면 어떻게 하나요?

매장 자체 콘텐츠는 해당 매장에서만 사용됩니다.
Community에 제출된 경우 HQ 승인 과정에서 검토됩니다.

### Q5: 긴급 공지를 모든 매장에 즉시 전달하려면?

1. 긴급 플레이리스트 생성
2. **필수 노출로 지정** (`isForced: true`)
3. 저장

필수 노출 콘텐츠는 모든 매장에서 즉시 적용됩니다.
단, **해당 콘텐츠에 대한 책임은 HQ에 있습니다**.

### Q6: 필수 노출 콘텐츠에 대해 매장에서 문의가 오면?

HQ/운영자가 직접 대응해야 합니다.
필수 노출 콘텐츠는 HQ 책임 영역입니다.
