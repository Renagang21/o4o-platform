# _investigations

이 폴더는 **조사 요청서 전용**입니다.

## 규칙

1. **파일명 형식**: `IR-YYYYMMDD-{topic}.md`
2. **생명주기**: 조사 완료 후 즉시 삭제
3. **Archive 이동**: 불가
4. **보관**: 불가

## 조사 요청서 헤더

모든 조사 요청서는 표준 헤더를 사용해야 합니다:

**헤더 템플릿**: `docs/_platform/work-order-headers.md` 참조

## 예시

```
docs/_investigations/IR-20251224-app-store-reliability.md
docs/_investigations/IR-20251224-auth-flow.md
```

## 중요

- 조사 내용은 자유 형식 (길이 제한 없음)
- 조사 완료 = 문서 삭제
- 요약만 반영 위치에 기록
