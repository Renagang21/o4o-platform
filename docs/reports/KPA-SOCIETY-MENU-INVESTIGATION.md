# 📄 조사 요청서

## 약사회 서비스 메뉴 정비를 위한 기능 조사

### 1. 조사 목적

* 약사회 서비스 메뉴에 현재 노출된 기능 중
  **메뉴 유지 불가 → 배너 전환 대상**을 식별한다.
* 본 조사는 **판단을 위한 조사**이며,
  메뉴 유지 여부에 대한 **결론은 이미 "메뉴에 두지 않는다"로 확정**되어 있다.

---

### 2. 조사 범위

* 대상:
  * kpa-society.co.kr 약사회 서비스 메뉴에 **현재 노출된 모든 기능**
* 제외:
  * 신규 기획 기능
  * 향후 아이디어
  * O4O 상업 서비스 내부 기능

---

### 3. 조사 기준 (판정 기준 고정)

각 기능에 대해 아래 체크리스트를 적용한다.
**하나라도 해당 시 → 메뉴 제거 / 배너 전환 확정**

#### A. 자금·거래

* 결제, 정산, 수수료, 가격 조건이 존재하는가
* 약사회 시스템이 자금 흐름에 관여하는가

#### B. 회원 유·불리

* 참여/미참여에 따른 유·불리가 발생하는가
* 참여가 사실상 압박 또는 기대 구조로 작동하는가

#### C. 사업 주체 오해

* 약사회가 사업을 주도·운영하는 것으로 인식될 여지가 있는가
* 약사회 명의로 사업 성과·조건이 설명되는가

#### D. 책임 오해

* 메뉴에 존재하는 것만으로 약사회 책임으로 오해될 수 있는가
* 분쟁 시 약사회 문의로 이어질 가능성이 있는가

## 📊 조사 결과

### 현재 약사회 서비스 메뉴 구조

**출처**: `services/web-kpa-society/src/components/Header.tsx` (Lines 23-78)

약사회 서비스는 현재 **6개 주요 메뉴**로 구성되어 있습니다:

1. **공지** (`/demo/news`)
   - 공지사항
   - 지부/분회 소식
   - 전체 약사회 소식
   - 갤러리
   - 보도자료

2. **포럼** (`/demo/forum`)
   - 전체 글
   - 글쓰기

3. **교육** (`/demo/lms`)
   - 교육 과정
   - 수료증

4. **공동구매** (`/demo/groupbuy`)
   - 진행중
   - 참여 내역

5. **자료실** (`/demo/docs`)
   - 서식/양식
   - 가이드라인
   - 규정/정관

6. **조직소개** (`/demo/organization`)
   - 약사회 소개
   - 지부/분회
   - 임원 안내
   - 연락처

### 조사 결론

#### 배너 전환 대상 (2개)

1. **교육** - 수강료, 수료증 발급 등 거래 요소 및 회원 유불리 발생 가능
2. **공동구매** - 결제·정산·가격 조건, 유불리, 사업 주체 오해, 책임 오해 모두 해당

#### 메뉴 유지 대상 (4개)

1. **공지** - 공공 정보 전달 기능
2. **포럼** - 회원 간 중립적 소통 공간
3. **자료실** - 공공 자료 제공
4. **조직소개** - 조직 정보 제공

### 권고사항

#### 1. 즉시 조치 필요

- **공동구매** 메뉴 제거 및 배너 전환
  - 배너 문구 예시: "회원 전용 공동구매 서비스 → O4O 플랫폼에서 이용"
  - 외부 링크 또는 별도 서비스로 안내

#### 2. 검토 후 조치

- **교육** 메뉴 검토
  - 현재 교육 서비스의 실제 운영 방식 확인 필요
  - 무료 공공 교육인 경우: 메뉴 유지 가능
  - 유료 교육 또는 외부 사업자 연계인 경우: 배너 전환

#### 3. 배너 전환 시 고려사항

- 배너 위치: 메인 페이지 또는 마이페이지 내 별도 섹션
- 배너 문구: 약사회 책임이 아님을 명확히 표시
- 연결 방식: 외부 링크 또는 별도 플랫폼으로 명확히 구분
