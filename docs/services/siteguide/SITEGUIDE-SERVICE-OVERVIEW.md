## 1. SiteGuide란?

SiteGuide는 **홈페이지를 가진 모든 사업자**를 위한 **AI 기반 사이트 안내 서비스**입니다.

### 핵심 정의

> **"SiteGuide는 방문자가 사이트를 이해하고 원하는 정보를 찾을 수 있도록 돕는 AI 안내 도우미입니다."**

### 서비스 성격

| 항목 | 설명 |
|------|------|
| **대상** | 홈페이지를 가진 모든 사업자 |
| **핵심 가치** | 방문자의 사이트 탐색 지원 |
| **AI 역할** | 전면 기능이 아닌 **보조 도구** |
| **운영 방식** | 다수 사업자가 동시 사용하는 **공용 서비스** |

---

## 2. 대상 사용자

### 2.1 1차 사용자: 사업자

SiteGuide를 **설치하고 관리하는 주체**입니다.

| 유형 | 예시 |
|------|------|
| 소규모 사업자 | 개인 쇼핑몰, 로컬 서비스 업체 |
| 중규모 사업자 | 전문 서비스 기업, B2B 업체 |
| 대규모 사업자 | 대형 포털, 엔터프라이즈 |

### 2.2 2차 사용자: 방문자

사업자의 홈페이지를 방문하여 **SiteGuide를 통해 안내받는 주체**입니다.

---

## 3. 제공 가치

### 3.1 사업자 관점

| 가치 | 설명 |
|------|------|
| **방문자 이탈 감소** | 필요한 정보를 빠르게 찾도록 지원 |
| **고객 문의 감소** | 반복 질문을 AI가 대응 |
| **사이트 활용도 향상** | 숨겨진 콘텐츠 노출 기회 증가 |

### 3.2 방문자 관점

| 가치 | 설명 |
|------|------|
| **빠른 정보 탐색** | 원하는 정보를 즉시 안내받음 |
| **자연어 질문** | 메뉴 탐색 없이 질문으로 해결 |
| **맥락적 도움** | 현재 페이지에 맞는 안내 제공 |

## 4. Neture/O4O와의 관계

### 4.1 명확한 분리

> **SiteGuide는 Neture의 하위 기능이 아닙니다.**

| 항목 | SiteGuide | Neture |
|------|-----------|--------|
| **대상** | 모든 사업자 | 약국/의료기관 |
| **도메인** | siteguide.co.kr | neture.co.kr |
| **종속 관계** | **독립** | 독립 |

### 4.2 공통점

- 동일 플랫폼(O4O Platform) 기반
- 동일 인프라(GCP Cloud Run) 사용
- 동일 인증 체계(Auth Core) 활용 가능

### 4.3 분리 원칙

```
┌─────────────────────────────────────────────────────────┐
│                    O4O Platform                          │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  ┌──────────────┐    ┌──────────────┐    ┌────────────┐ │
│  │   Neture     │    │  SiteGuide   │    │   Others   │ │
│  │  (약국 전용)  │    │ (모든 사업자) │    │   ...      │ │
│  └──────────────┘    └──────────────┘    └────────────┘ │
│         │                   │                  │         │
│         └───────────────────┴──────────────────┘         │
│                         │                                │
│              Shared Infrastructure                       │
│              (Auth, Infra, AI Core)                      │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## 5. 책임 경계

### 5.1 SiteGuide가 책임지는 것

| 항목 | 설명 |
|------|------|
| AI 안내 기능 | 방문자 질문 응답 |
| 사이트 컨텍스트 수집 | 사업자 사이트 정보 인덱싱 |
| 기본 통계 제공 | 최소한의 사용 현황 |
| 위젯 제공 | 사업자 사이트에 삽입하는 UI |

### 5.2 SiteGuide가 책임지지 않는 것

| 항목 | 소유권 |
|------|--------|
| 사업자 사이트 콘텐츠 | 사업자 |
| AI 응답의 정확성 보장 | 면책 (참고 정보) |
| 비즈니스 성과(매출, 전환) | 사업자 |
| 사업자 사이트 가용성 | 사업자 |

### 5.3 책임 분기 선언

> **SiteGuide는 AI 안내 기능을 제공하며,**
> **응답 내용의 정확성 및 비즈니스 성과에 대해 보장하지 않습니다.**
> **사이트 콘텐츠의 적절성과 정확성은 사업자의 책임입니다.**

## 6. AI의 역할

### 6.1 AI는 보조 도구

SiteGuide에서 AI는 **전면 기능이 아닌 보조 도구**입니다.

| 구분 | 설명 |
|------|------|
| **주체** | 방문자 (사이트 탐색 주체) |
| **AI 역할** | 안내 도우미 (질문 응답) |
| **책임** | AI 응답 선택/활용 책임은 방문자 |

### 6.2 AI 기능 범위 (예정)

| 기능 | 설명 | 상태 |
|------|------|------|
| 사이트 안내 | 페이지/콘텐츠 안내 | 예정 |
| 질문 응답 | 자연어 질문 대응 | 예정 |
| 통계 해석 | 사용 현황 요약 | 예정 |

### 6.3 AI Core와의 관계

SiteGuide의 AI 기능은 **AI Core의 정책과 계약을 준수**합니다.

```
AI Core (정책/계약/로그 기준)
    │
    ├── SiteGuide AI (서비스 레이어 - AI 사용)
    │
    └── 기타 서비스 AI ...
```

## 7. 통계 제공 원칙

### 7.1 최소 통계만 제공

SiteGuide는 **최소한의 통계만 제공**합니다.

| 제공 | 미제공 |
|------|--------|
| 질문 수 | 상세 분석 |
| 응답 성공률 | 비즈니스 인사이트 |
| 사용 추이 | KPI 보장 |

### 7.2 해석은 AI에게 위임

복잡한 통계 해석은 **AI가 요약/설명**하는 방식으로 제공합니다.

> "지난 주 질문 수가 15% 증가했습니다. 주로 '배송' 관련 질문이 많았습니다."
