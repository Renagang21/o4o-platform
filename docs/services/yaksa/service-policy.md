# Yaksa Branch Service - Policy

> 서비스 정책 문서

## 접근 정책

### 회원 가입
- 약사 면허 검증 필수
- 조직 승인 필요

### 권한 관리
- **Admin**: 전체 관리
- **Organization Admin**: 조직 관리, 회원 승인
- **Educator**: 교육 과정 관리
- **Member**: 포럼 참여, 교육 수강, 보고서 제출

## 데이터 정책

### 데이터 소유권
- 회원 정보: 조직 소유
- 교육 기록: 조직 소유
- 보고서: 조직 소유

### 데이터 보존
- 회원 정보: 탈퇴 후 5년
- 교육 기록: 영구 보존
- 보고서: 영구 보존

## 비즈니스 정책

### 연회비 정책
- 연회비: 조직별 설정
- 납부 기한: 매년 1월 31일
- 미납 시: 서비스 제한

### 교육 정책
- 필수 교육 시간: 연간 8시간
- 수료증 발급: 자동
- 교육 기록: 영구 보존

## 기술 정책

### API 사용 제한
- Rate limit: 500 req/hour (회원)
- 외부 API 접근: 제한

### 통합 정책
- 외부 교육 기관 연동: 제한적 허용
- 정부 시스템 연동: 허용

### AI 사용 범위
- 포럼 추천 알고리즘
- 교육 과정 추천
