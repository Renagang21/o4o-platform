# Cosmetics Retail Service - Policy

> 서비스 정책 문서

## 접근 정책

### 회원 가입
- 일반 소비자: 이메일 인증
- 판매자/공급사: 사업자 등록 확인
- 파트너/인플루언서: 신청 후 승인

### 권한 관리
- **Admin**: 전체 관리
- **Seller**: 상품 등록, 주문 관리, 정산 조회
- **Supplier**: Offer 관리, 재고 관리, 정산 조회
- **Partner**: 추천 링크 생성, 커미션 조회
- **Customer**: 구매, 리뷰 작성

## 데이터 정책

### 데이터 소유권
- 제품 정보: Supplier 소유
- 주문 정보: 플랫폼 소유 (ecommerce-core)
- 고객 정보: 플랫폼 소유

### 데이터 보존
- 주문 데이터: 영구 보존
- 고객 데이터: 탈퇴 후 1년
- 정산 데이터: 5년

## 비즈니스 정책

### 수수료 정책
- 플랫폼 수수료: 10% (기본)
- 파트너 커미션: 5% (기본)
- 정산 주기: 월 1회

### 거래 정책
- Seller 승인 필요
- Supplier 승인 필요
- 최소 주문 금액: 없음
- 환불 기간: 7일

## 기술 정책

### API 사용 제한
- Rate limit: 1000 req/hour (인증된 사용자)
- Rate limit: 100 req/hour (비인증 사용자)

### 통합 정책
- 외부 PG 연동 허용
- 외부 물류사 연동 허용
- 외부 ERP 연동 허용 (Supplier)

### AI 사용 범위
- 제품 추천 알고리즘
- 피부 타입 분석
- 파트너 콘텐츠 생성 지원 (partner-ai-builder)
