# 폴더 구조 및 역할 (2024-06-18 기준)

```
o4o-platform/
├── docs/                # 프로젝트 문서
├── services/            # 서비스별 소스코드
│   ├── api-server/      # Express API 서버
│   └── main-site/       # React 웹앱
├── scripts/             # 자동화 스크립트
└── tests/               # E2E 테스트
```

- **docs/**: 실제 개발/운영/AI 협업 중심 문서
- **services/**: 백엔드(API), 프론트엔드(main-site) 등 서비스별 코드
- **scripts/**: 코드/배포/테스트 자동화 스크립트
- **tests/**: E2E, 통합, 단위 테스트 코드

> 상세 폴더/설정파일 구조는 추후 점검 및 업데이트 예정
