## 드롭쉬핑이란?

**드롭쉬핑(Dropshipping)**은 판매자가 재고를 보유하지 않고 상품을 판매하는 비즈니스 모델입니다.

### 기본 흐름

```
고객 → 판매자 → 공급자 → 배송 → 고객
         ↓
      주문 접수
```

1. **고객**이 판매자 사이트에서 상품 구매
2. **판매자**가 공급자에게 주문 전달
3. **공급자**가 직접 고객에게 배송
4. **판매자**는 마진(공급가와 판매가 차액)을 수익으로 확보

### 장점

✅ **낮은 초기 비용**: 재고 구매 없이 시작
✅ **재고 리스크 제로**: 판매된 만큼만 구매
✅ **확장 용이**: 물리적 제약 없이 상품 추가
✅ **유연한 운영**: 시간과 장소에 구애받지 않음

## O4O 드롭쉬핑 구조

O4O 플랫폼은 **3자 구조**로 운영됩니다:

### 1️⃣ 공급자 (Supplier)

**역할:** 상품을 등록하고 드롭쉬핑 네트워크에 공급

**주요 업무:**
- 상품 등록 및 관리
- 공급가, MSRP, 파트너 수수료율 설정
- 재고 관리
- 주문 처리 및 배송
- 판매자/파트너에게 상품 정보 제공

**수익 구조:**
```
공급가 × 판매 수량 = 수익
```

---

### 2️⃣ 판매자 (Seller)

**역할:** 공급자의 상품을 자신의 가격으로 재판매

**주요 업무:**
- 공급자 상품 선택
- 판매가 자율 설정
- 마케팅 및 고객 응대
- 주문 관리
- 마진 정산 수령

**수익 구조:**
```
(판매가 - 공급가) × 판매 수량 = 마진
```

---

### 3️⃣ 파트너 (Partner/Affiliate)

**역할:** 제휴 마케팅을 통해 트래픽 유도 및 수수료 수령

**주요 업무:**
- 개인화된 추천 링크 생성
- SNS, 블로그, 유튜브 등에서 상품 홍보
- 클릭 및 전환 추적
- 수수료 정산 수령

**수익 구조:**
```
판매가 × 수수료율(%) × 전환 수 = 수수료
```

---

### 수익 분배 예시

**상품 판매가:** 100,000원
**공급가:** 70,000원
**파트너 수수료율:** 10%

| 역할 | 수익 | 계산 |
|------|------|------|
| 공급자 | 70,000원 | 공급가 |
| 판매자 | 20,000원 | 판매가 - 공급가 - 파트너 수수료 |
| 파트너 | 10,000원 | 판매가 × 10% |

## 시작하기

### 1단계: 회원가입

1. O4O 플랫폼 회원가입
2. 이메일 인증 완료
3. 기본 프로필 작성

### 2단계: 역할 선택 및 인증

**공급자 신청 페이지:**
```
페이지에 다음 shortcode 삽입:
[role_verification type="supplier"]
```

**판매자 신청 페이지:**
```
[role_verification type="seller"]
```

**파트너 신청 페이지:**
```
[role_verification type="partner"]
```

관리자 승인 후 해당 역할의 대시보드 및 기능에 접근할 수 있습니다.

## 공급자 가이드

공급자는 O4O 드롭쉬핑 네트워크의 **상품 공급원**입니다.

### 공급자 대시보드 설정

**대시보드 페이지 생성:**
1. 워드프레스 관리자 → 페이지 → 새 페이지 추가
2. 제목: "공급자 대시보드"
3. 본문에 shortcode 삽입:
```
[supplier_dashboard]
```
4. 발행

**표시 내용:**
- 등록 상품 현황
- 총 판매액 및 정산 금액
- 승인 대기 상품
- 파트너별 판매 통계

---

### 상품 등록하기

#### 방법 1: 상품 편집기 페이지 사용

**페이지 생성:**
1. 새 페이지 추가
2. 제목: "상품 등록"
3. Shortcode 삽입:
```
[supplier_product_editor mode="create"]
```

#### 방법 2: 관리자 대시보드 사용

1. 워드프레스 관리자 → 드롭쉬핑 → 상품 관리
2. "새 상품 추가" 클릭
3. 상품 정보 입력

#### 입력 항목

| 항목 | 설명 | 필수 |
|------|------|------|
| **상품명** | 상품의 공식 명칭 | ✅ |
| **설명** | 상세 설명, 스펙 | ✅ |
| **카테고리** | 상품 분류 | ✅ |
| **이미지** | 대표 이미지 및 갤러리 | ✅ |
| **공급가** | 공급자가 받는 금액 | ✅ |
| **MSRP** | 권장 소비자 가격 | ✅ |
| **파트너 수수료율** | 제휴 마케팅 수수료 (%) | ✅ |
| **재고 수량** | 현재 재고 | ✅ |
| **배송 정보** | 배송비, 배송 기간 | ✅ |
| **SKU** | 재고 관리 코드 | |
| **무게/크기** | 배송비 계산용 | |

---

### 상품 관리하기

**상품 목록 페이지 생성:**
```
[supplier_products limit="20"]
```

**주요 기능:**
- 상품 편집/삭제
- 재고 일괄 업데이트
- 가격 변경
- 판매 통계 확인

**재고 관리 팁:**
- 실시간 재고 동기화 활성화
- 품절 시 자동 숨김 설정
- 재고 알림 설정 (최소 재고 도달 시 알림)

---

### 주문 처리하기

1. **주문 알림 수신**
   - 이메일 또는 대시보드 알림

2. **주문 확인**
   - 주문 상세 정보 확인
   - 배송 주소, 상품, 수량 체크

3. **포장 및 배송**
   - 판매자가 아닌 **최종 고객 정보**로 배송
   - 송장 번호 시스템에 입력

4. **배송 완료 처리**
   - 배송 완료 후 상태 업데이트
   - 정산 대기 상태로 전환

---

### 정산 받기

**정산 주기:** 월 1회 (매월 15일)

**정산 조건:**
- 배송 완료 + 7일 경과
- 반품 기간 종료
- 고객 결제 확정

**정산 확인:**
```
[supplier_analytics type="revenue"]
```

## 판매자 가이드

판매자는 공급자의 상품을 **자신의 브랜드/가격**으로 재판매합니다.

### 판매자 대시보드 설정

**대시보드 페이지 생성:**
```
[seller_dashboard]
```

**표시 내용:**
- 총 판매액 및 순이익
- 마진율 통계
- 베스트셀러 상품
- 주문 현황

---

### 상품 선택하기

#### 상품 마켓플레이스 페이지

**페이지 생성:**
1. 새 페이지 추가
2. 제목: "상품 찾기"
3. Shortcode 삽입:
```
[seller_products]
```

**상품 선택 기준:**
- 수요가 높은 카테고리
- 마진율이 높은 상품
- 배송이 빠른 공급자
- 품질이 검증된 상품

---

### 판매가 설정하기

#### 가격 관리 페이지

```
[seller_pricing_manager]
```

#### 가격 설정 전략

**1. 원가 기반 가격 책정**
```
판매가 = 공급가 + 희망 마진 + 파트너 수수료
```

**예시:**
- 공급가: 70,000원
- 희망 마진: 20,000원
- 파트너 수수료 (10%): 10,000원
- **판매가: 100,000원**

**2. 시장 가격 조사**
- 경쟁사 가격 확인
- MSRP 참고
- 고객 지불 의향 분석

**3. 마진율 계산**
```
마진율 = (판매가 - 공급가 - 파트너 수수료) / 판매가 × 100
```

**건강한 마진율:** 20-40%

---

### 마케팅 및 판매

#### 상품 페이지 최적화

**SEO 최적화:**
- 키워드가 포함된 상품명
- 상세하고 매력적인 설명
- 고품질 이미지
- 고객 리뷰 활성화

**상품 상세 페이지 구성:**
```html
<!-- 상품 정보 -->
[product id="123" show_description="true"]

<!-- 추가 상품 -->
<h3>이런 상품은 어때요?</h3>
[product_grid category="전자제품" limit="4"]

<!-- 문의 폼 -->
<h3>궁금한 점이 있으신가요?</h3>
[form id="product-inquiry"]
```

### 주문 관리

**주문 흐름:**

1. **고객 주문 접수**
   - 자동으로 공급자에게 전달
   - 고객에게 주문 확인 이메일 발송

2. **공급자 배송 진행**
   - 송장 번호 자동 업데이트
   - 고객에게 배송 시작 알림

3. **배송 추적**
   - 실시간 배송 상태 확인
   - 고객 문의 대응

4. **배송 완료**
   - 고객 만족도 확인
   - 리뷰 요청

**고객 응대 팁:**
- 빠른 응답 (24시간 이내)
- 친절하고 전문적인 커뮤니케이션
- 문제 발생 시 즉시 공급자와 협의
- 교환/반품 정책 명확히 안내

---

### 정산 받기

**정산 페이지:**
```
[seller_settlement period="30d"]
```

**정산 내역:**
- 정산 예정 금액
- 정산 완료 내역
- 거래별 수수료
- 지급 일정

**정산 주기:** 월 2회 (1일, 15일)

**정산 조건:**
- 배송 완료 확인
- 반품 기간 종료 (7일)
- 공급자 정산 완료

## 파트너 가이드

파트너는 **제휴 마케팅**을 통해 트래픽을 유도하고 수수료를 받습니다.

### 파트너 대시보드 설정

**대시보드 페이지 생성:**
```
[partner_dashboard]
```

**표시 내용:**
- 총 수익 및 전환율
- 개인 추천 링크
- 최근 클릭 및 전환 통계
- 월별 수익 차트

---

### 추천 링크 생성하기

#### 링크 생성기 페이지

```
[partner_link_generator]
```

#### 링크 생성 방법

1. **상품 선택**
   - 홍보할 상품 검색
   - 수수료율 확인

2. **캠페인 태그 추가 (선택)**
   ```
   ?campaign=youtube_review
   ?campaign=instagram_story
   ?campaign=blog_post_march
   ```

3. **링크 복사**
   ```
   https://example.com/product/123?ref=partner_abc&campaign=youtube
   ```

4. **추적 코드 활성화**
   - 클릭 수, 전환율, 수익 자동 추적

---

### 홍보 상품 선택하기

**상품 목록 페이지:**
```
[partner_products sortBy="commission" limit="20"]
```

#### 상품 선택 기준

✅ **높은 수수료율**: 10% 이상
✅ **인기 상품**: 전환율이 높은 상품
✅ **품질 검증**: 리뷰 평점 4.0 이상
✅ **경쟁력 있는 가격**: 시장 가격 대비 합리적
✅ **빠른 배송**: 고객 만족도 향상

#### 정렬 옵션

| 정렬 기준 | 설명 |
|----------|------|
| `commission` | 수수료율 높은 순 |
| `performance` | 전환율 높은 순 |
| `price` | 가격 낮은 순 |
| `newest` | 최신 등록 순 |

#### 추적 지표

| 지표 | 설명 | 개선 방법 |
|------|------|----------|

## 정산 시스템

### 정산 흐름도

```
주문 완료 → 배송 완료 → 반품 기간 종료 (7일) → 정산 대기 → 정산 지급
```

### 역할별 정산 일정

| 역할 | 정산 주기 | 정산일 | 최소 금액 |
|------|----------|--------|----------|
| **공급자** | 월 1회 | 매월 15일 | 100,000원 |
| **판매자** | 월 2회 | 1일, 15일 | 50,000원 |
| **파트너** | 월 1회 | 매월 말일 | 50,000원 |

### 정산 금액 계산

#### 공급자 정산
```
정산 금액 = Σ(공급가 × 판매 수량) - 플랫폼 수수료(3%)
```

#### 판매자 정산
```
정산 금액 = Σ(판매가 - 공급가 - 파트너 수수료) - 플랫폼 수수료(5%)
```

#### 파트너 정산
```
정산 금액 = Σ(판매가 × 수수료율) - 플랫폼 수수료(10%)
```

### 플랫폼 수수료

| 역할 | 수수료율 | 설명 |
|------|---------|------|
| 공급자 | 3% | 플랫폼 이용료 |
| 판매자 | 5% | 거래 수수료 |
| 파트너 | 10% | 제휴 관리 수수료 |

## FAQ

### 일반 질문

**Q1. 여러 역할을 동시에 할 수 있나요?**
A. 네, 가능합니다. 예를 들어 공급자이면서 동시에 다른 공급자의 상품을 판매자로 재판매할 수 있습니다.

**Q2. 역할 변경이나 추가는 어떻게 하나요?**
A. 역할 인증 페이지에서 추가 역할을 신청하고 관리자 승인을 받으면 됩니다.

**Q3. 수수료는 협상 가능한가요?**
A. 일정 거래량 이상 시 개별 협상 가능합니다. 관리자에게 문의하세요.

---

### 공급자 질문

**Q4. 상품 승인이 거부되면 어떻게 하나요?**
A. 거부 사유를 확인하고 수정 후 재신청할 수 있습니다. 일반적인 거부 사유:
- 불완전한 상품 정보
- 저품질 이미지
- 불합리한 가격 책정
- 금지된 카테고리

**Q5. 재고 관리는 자동으로 되나요?**
A. 네, 판매 시 자동으로 재고가 차감됩니다. 외부 ERP와 연동도 가능합니다.

**Q6. 반품/교환은 어떻게 처리하나요?**
A. 공급자가 직접 처리합니다. 판매자가 고객 요청을 받아 공급자에게 전달하면, 공급자가 반품 주소 안내 및 교환품 발송을 진행합니다.

---

### 판매자 질문

**Q7. 가격은 얼마든지 자유롭게 설정할 수 있나요?**
A. 기본적으로 자유롭지만, 과도하게 높은 가격은 관리자 검토 대상이 될 수 있습니다. MSRP의 150% 이내를 권장합니다.

**Q8. 공급자가 여러 명이면 어떻게 하나요?**
A. 상품별로 다른 공급자를 선택할 수 있습니다. 각 주문은 해당 공급자에게 자동으로 전달됩니다.

**Q9. 마진이 마이너스가 될 수 있나요?**
A. 시스템에서 공급가+파트너 수수료보다 낮은 가격 설정을 방지합니다.

---

### 파트너 질문

**Q10. 어떤 상품을 홍보해야 수익이 높나요?**
A. 다음 조건을 만족하는 상품을 선택하세요:
- 수수료율 10% 이상
- 전환율 3% 이상
- 판매가 50,000원 이상 (높은 절대 수익)
- 자신의 채널 구독자와 관련성 높음

**Q11. 링크를 무단으로 복사해 쓰면 어떻게 되나요?**
A. 각 링크에는 고유한 파트너 코드가 포함되어 있어, 누가 복사해도 수익은 원래 파트너에게 귀속됩니다.

**Q12. 수수료는 언제까지 유효한가요?**
A. 일반적으로 링크 클릭 후 30일 이내 구매 시 수수료가 인정됩니다. (쿠키 유효 기간)

## 문제 해결

### 상품 관련

**문제: 상품이 목록에 표시되지 않습니다.**

**원인 및 해결:**
- ✅ 상품 상태가 "승인됨"인지 확인
- ✅ 재고가 0이 아닌지 확인
- ✅ 공개 상태가 "공개"인지 확인
- ✅ 카테고리가 올바르게 설정되었는지 확인

---

**문제: 가격이 자동으로 계산되지 않습니다.**

**원인 및 해결:**
- ✅ 공급가가 입력되었는지 확인
- ✅ 파트너 수수료율이 설정되었는지 확인
- ✅ 브라우저 캐시 삭제 후 재시도

---

### 주문 관련

**문제: 주문이 공급자에게 전달되지 않습니다.**

**원인 및 해결:**
- ✅ 공급자 이메일이 올바른지 확인
- ✅ 스팸 폴더 확인 요청
- ✅ 시스템 알림 설정 확인
- ✅ 관리자에게 문의

---

**문제: 배송 정보가 업데이트되지 않습니다.**

**원인 및 해결:**
- ✅ 송장 번호 형식이 올바른지 확인
- ✅ 택배사가 지원 목록에 있는지 확인
- ✅ 배송 추적 API 연동 상태 확인

---

### 정산 관련

**문제: 정산 금액이 예상과 다릅니다.**

**원인 및 해결:**
- ✅ 플랫폼 수수료가 차감되었는지 확인
- ✅ 반품/취소된 주문이 있는지 확인
- ✅ 정산 페이지에서 상세 내역 확인
- ✅ 정산 조건 (배송 완료 + 7일) 충족 확인

---

**문제: 정산이 지연되고 있습니다.**

**원인 및 해결:**
- ✅ 정산 조건을 모두 충족했는지 확인
- ✅ 최소 인출 금액 이상인지 확인
- ✅ 은행 계좌 정보가 올바른지 확인
- ✅ 공휴일/주말로 인한 지연 여부 확인
- ✅ 지연 시 관리자에게 문의

---
