## 3. KPI별 즉시 판단 기준

### 전환율 (샘플→구매)

| 수치 | 상태 | 행동 |
|------|------|------|
| ≥15% | 🟢 정상 | 유지 |
| 10~15% | ⚪ 주의 | 진열 점검 |
| <10% | 🔴 위험 | 위치/멘트 변경 |
| <5% | 🔴 심각 | 제품 교체 검토 |

### 건강 점수 (진열)

| 수치 | 상태 | 행동 |
|------|------|------|
| ≥80% | 🟢 정상 | 유지 |
| 50~80% | ⚪ 주의 | 미인증 확인 |
| <50% | 🔴 위험 | 즉시 인증 처리 |

### 재고 부족

| 수치 | 상태 | 행동 |
|------|------|------|
| 0개 | 🟢 정상 | 유지 |
| 1~2개 | ⚪ 주의 | 보충 계획 |
| ≥3개 | 🔴 위험 | 즉시 보충 요청 |

### 미인증 진열

| 수치 | 상태 | 행동 |
|------|------|------|
| 0개 | 🟢 정상 | 유지 |
| 1~2개 | ⚪ 주의 | 당일 처리 |
| ≥3개 | 🔴 위험 | 즉시 인증 |

## 4. 상황별 빠른 대응

### 🔴 재고 부족 발생 시

```
1. Supplier 승인 상태 확인
2. 승인됨 → 보충 요청
3. 미승인 → 진열 비활성화
```

### 🔴 전환율 하락 시

```
1. 진열 위치 확인 (눈높이?)
2. 페이싱 확인 (최소 2개?)
3. 체험 멘트 변경
4. 1주 후 재확인
```

### 🔴 미인증 진열 발견 시

```
1. 사진 등록 여부 확인
2. 사진 없음 → 촬영 후 등록
3. 사진 있음 → 인증 처리
4. 24시간 내 완료
```

## 5. 일일 3회 체크 타임

| 시간 | 체크 항목 |
|------|----------|
| **오전** (개점 후 30분) | 재고 부족, 미인증, 전일 전환율 |
| **오후** (14:00~15:00) | 사용량 추이, 보충 필요 |
| **마감** (폐점 전 30분) | 금일 KPI, 내일 조치 사항 |

## 6. 핵심 원칙

> **적색이 보이면 즉시 행동한다.**
> **회색이 2일 연속이면 조치한다.**
> **녹색은 건드리지 않는다.**
