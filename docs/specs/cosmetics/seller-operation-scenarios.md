## 1. 문서 목적

이 문서는 **Cosmetics Seller Mode의 운영 기준을 고정**한다.

목적:
- 운영 상황별 판단 기준 명확화
- KPI 해석 방식 통일
- 즉시 행동 가이드 제공
- **운영 일관성과 재현성 확보**

이 문서가 정의하지 않는 것:
- 신규 기능 추가 ❌
- 로직 변경 ❌
- UI 대개편 ❌

## 2. 핵심 운영 시나리오

### 시나리오 1. 신규 샘플 입고 발생

| 항목 | 내용 |
|------|------|
| **상황** | 신규 제품 샘플 입고, Sample Inventory 증가 |
| **확인 KPI** | 총 재고(품목 수), 최소 재고 기준 충족 여부 |
| **즉시 행동** | 샘플 진열(Display) 등록, 체험용/테스터 구분 확인 |
| **후속 조치** | 3~5일 내 Sample Usage 확인 |

**판단 기준**
```
IF 신규 입고 완료
  → Display 등록 필수
  → 체험용/테스터 구분 확인
  → 3~5일 후 Usage 데이터 확인
```

---

### 시나리오 2. 재고 부족/품절 발생

| 항목 | 내용 |
|------|------|
| **상황** | 재고 부족 또는 품절 표시 |
| **확인 KPI** | 부족/품절 품목 수, 최근 사용량 추이 |
| **즉시 행동** | Supplier 승인 상태 확인, 샘플 보충 요청 또는 진열 비활성화 |
| **후속 조치** | 보충 완료 후 Display 재활성화 |

**판단 기준**
```
IF 재고 부족 (lowStockCount > 0)
  → Supplier 승인 상태 확인
  → 승인됨: 보충 요청
  → 미승인: Display 비활성화

IF 품절 (outOfStockCount > 0)
  → Display 즉시 비활성화
  → 보충 후 재활성화
```

---

### 시나리오 3. 샘플 사용량 증가 (체험 활성)

| 항목 | 내용 |
|------|------|
| **상황** | 특정 제품 샘플 사용 급증 |
| **확인 KPI** | 샘플 사용량(일/주), 체험 고객 수 |
| **즉시 행동** | 재고 소진 속도 점검, 진열 위치 유지 또는 확대 |
| **후속 조치** | 전환율(구매) 확인 |

**판단 기준**
```
IF 사용량 급증 (전주 대비 +30% 이상)
  → 재고 소진 예상일 계산
  → 소진 예상 < 7일: 보충 요청
  → 진열 위치 확대 검토
  → 1주 후 전환율 확인
```

---

### 시나리오 4. 전환율 하락

| 항목 | 내용 |
|------|------|
| **상황** | 샘플 사용 대비 구매 전환율 저하 |
| **확인 KPI** | Sample → Order 전환율, Display 건강 점수 |
| **즉시 행동** | 진열 위치/페이싱 점검, 체험 멘트/설명 방식 변경 |
| **후속 조치** | 1주 단위 전환율 재확인 |

**판단 기준**
```
IF 전환율 < 10%
  → 진열 위치 점검 (눈높이 여부)
  → 페이싱 수량 확인 (최소 2개)
  → 체험 멘트 변경 검토

IF 전환율 < 5%
  → 제품 교체 검토
  → 또는 진열 중단 검토
```

---

### 시나리오 5. 미인증 진열 발생

| 항목 | 내용 |
|------|------|
| **상황** | 미인증 Display 존재 |
| **확인 KPI** | 미인증 진열 수, 진열 건강 점수 |
| **즉시 행동** | 진열 인증 처리, 사진 등록 여부 확인 |
| **후속 조치** | 인증 후 KPI 변화 확인 |

**판단 기준**
```
IF 미인증 진열 존재 (unverifiedDisplays > 0)
  → 사진 등록 확인
  → 사진 없음: 촬영 후 등록
  → 사진 있음: 인증 처리
  → 24시간 내 처리 완료 목표
```

## 3. Seller Dashboard KPI 해석 가이드

### 3.1 KPI 색상 해석 기준

| 색상 | 의미 | 행동 |
|------|------|------|
| **Positive (녹색)** | 정상 범위 | 현재 상태 유지 |
| **Neutral (회색)** | 관찰 필요 | 모니터링 강화 |
| **Negative (적색)** | 즉시 조치 필요 | 당일 내 조치 |

### 3.2 기간 필터 사용 원칙

| 기간 | 용도 | 판단 기준 |
|------|------|----------|
| **오늘** | 현장 즉시 판단 | 당일 문제 발견 및 조치 |
| **주간** | 운영 패턴 확인 | 추세 분석, 보충 계획 |
| **월간** | 제품 유지/교체 판단 | 전략적 의사결정 |

### 3.3 주요 KPI 임계값

| KPI | 정상 | 주의 | 위험 |
|-----|------|------|------|
| 전환율 | ≥15% | 10~15% | <10% |
| 건강 점수 | ≥80% | 50~80% | <50% |
| 재고 부족 | 0개 | 1~2개 | ≥3개 |
| 미인증 진열 | 0개 | 1~2개 | ≥3개 |

## 5. 운영 기준 고정 선언

> **본 문서 확정일(2025-12-17) 기준으로:**
>
> 1. Cosmetics Seller Mode는 "기능 개발 단계"를 공식 종료한다.
> 2. 이후 신규 개발은 "운영 데이터 기반"으로만 진행한다.
> 3. 본 문서의 시나리오와 KPI 기준은 변경 없이 6개월간 유지한다.
> 4. 운영 중 발견된 개선점은 별도 Work Order로 처리한다.
> 5. 본 기준을 따르지 않는 운영 행위는 비표준으로 간주한다.
