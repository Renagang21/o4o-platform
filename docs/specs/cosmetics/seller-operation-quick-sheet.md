## 📊 KPI 빠른 판단

### 전환율 (샘플→구매)
- 🟢 **15% 이상** → 유지
- ⚪ **10~15%** → 진열 점검
- 🔴 **10% 미만** → 위치/멘트 변경

### 건강 점수 (진열)
- 🟢 **80% 이상** → 유지
- ⚪ **50~80%** → 미인증 확인
- 🔴 **50% 미만** → 즉시 인증 처리

### 재고 부족
- ⚪ **0개** → 유지
- ⚪ **1~2개** → 보충 계획
- 🔴 **3개 이상** → 즉시 보충

### 미인증 진열
- ⚪ **0개** → 유지
- ⚪ **1~2개** → 당일 처리
- 🔴 **3개 이상** → 즉시 인증

## ⏰ 일일 3회 체크

| 시간 | 체크 항목 |
|:----:|----------|
| **오전** | 재고 부족, 미인증, 전일 전환율 |
| **오후** | 사용량 추이, 보충 필요 |
| **마감** | 금일 KPI, 내일 조치 사항 |

---

## 🆘 상황별 즉시 대응

### 🔴 재고 부족
```
1. Supplier 승인 확인
2. 승인 → 보충 요청
3. 미승인 → 진열 비활성화
```

### 🔴 전환율 하락
```
1. 진열 위치 확인 (눈높이?)
2. 페이싱 확인 (최소 2개?)
3. 체험 멘트 변경
```

### 🔴 미인증 진열
```
1. 사진 등록 확인
2. 없으면 → 촬영 후 등록
3. 있으면 → 인증 처리
```
