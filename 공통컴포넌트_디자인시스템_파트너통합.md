# o4o-Platform 공통 컴포넌트 및 디자인 시스템 가이드 (파트너 시스템 통합)

## 📋 개요

o4o-platform의 **공통 디자인 시스템**은 **4-Way 생태계**(공급자-판매자-파트너-구매자)와 **등급별 차별화 서비스**(VIP/Premium/Member/B2B)를 지원하는 통합 플랫폼입니다. **정보 중심 제품의 신뢰도 기반 거래**와 **법적 준수를 기반으로 한 건전한 파트너십**을 통해 일관된 사용자 경험을 제공합니다.

### 핵심 설계 원칙 (확장)
1. **신뢰성 우선**: 모든 UI 요소는 신뢰도를 강화하는 방향으로 설계
2. **정보 계층화**: 복잡한 정보를 단계적으로 제공하는 Progressive Disclosure
3. **역할별 최적화**: 공급자, 판매자, 파트너, 구매자별 맞춤형 인터페이스
4. **등급별 차별화**: VIP/Premium/Member/B2B 등급에 따른 차등 서비스
5. **투명성 보장**: 모든 데이터와 프로세스의 투명한 공개
6. **법적 준수**: 파트너 시스템의 완전한 법적 준수 (수수료 35% 미만, 단일 계층)
