# Admin.Management Theme Profile

**한글 명칭**: 관리자·운영형
**영문 기준 명칭**: `Admin.Management`

---

## 사용 목적

운영자, 관리자, 내부 사용자가 **대량의 데이터를 효율적으로 관리**하는 화면에 적용한다.

적용 대상:
- Admin Dashboard
- 백오피스 관리 화면
- 데이터 테이블 중심 CRUD
- 시스템 설정/구성 화면

---

## UI 판단 기준

### 정보 밀도
- **높음** - 한 화면에 많은 정보를 표시
- 테이블, 리스트, 필터가 주요 컴포넌트
- 스크롤보다 페이지네이션 선호

### 여백
- **최소화** - 공간 효율 우선
- 컴포넌트 간 간격: 8-16px
- 섹션 간 간격: 16-24px

### 색상 사용
- **기능 중심** - 상태 표시, 액션 구분에 집중
- 배경: 흰색 또는 연한 회색
- 강조: 시스템 Primary 색상 (일반적으로 Blue 계열)
- 상태 색상: Success(Green), Warning(Yellow), Error(Red), Info(Blue)

### 톤
- **실용적, 전문적**
- 장식 요소 최소화
- 아이콘은 기능 전달용으로만 사용
