# Service.Professional Theme Profile

**한글 명칭**: 서비스·전문가형
**영문 기준 명칭**: `Service.Professional`

---

## 사용 목적

전문가(약사, 파트너, 공급자 등)가 **업무를 수행하면서도 서비스 경험을 느낄 수 있는** 화면에 적용한다.

적용 대상:
- 약국 대시보드 (Pharmacy Dashboard)
- 파트너/공급자 포털
- 전문가용 서비스 화면
- 신청/승인 플로우

---

## UI 판단 기준

### 정보 밀도
- **중간** - 핵심 정보 우선, 세부 정보는 계층화
- 카드 기반 레이아웃
- 요약 → 상세 드릴다운 패턴

### 여백
- **균형** - 가독성과 효율성 사이
- 컴포넌트 간 간격: 16-24px
- 섹션 간 간격: 24-32px

### 색상 사용
- **브랜드 중심** - Primary 색상 적극 활용
- 카드/섹션에 그라데이션 허용 (절제하여)
- 상태 표시는 Admin과 동일한 시스템 색상

### 톤
- **신뢰감, 전문성**
- 깔끔하지만 차가움을 피함
- 사용자를 안내하는 친절한 UX 문구 허용

## 하지 말아야 할 것

| 금지 | 이유 |
|------|------|
| Admin처럼 빽빽한 테이블 | 서비스 경험 저하 |
| Consumer처럼 과한 마케팅 요소 | 업무 맥락과 부적합 |
| 복잡한 다단 레이아웃 | 핵심 정보 흐림 |
| 아이콘 없는 텍스트만의 메뉴 | 직관성 저하 |
| 불필요한 장식적 일러스트 | 전문성 훼손 |
| 너무 밝거나 화려한 색상 | 업무 집중 방해 |
