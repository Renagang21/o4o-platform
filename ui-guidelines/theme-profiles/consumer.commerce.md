# Consumer.Commerce Theme Profile

**한글 명칭**: 소비자·커머스형
**영문 기준 명칭**: `Consumer.Commerce`

---

## 사용 목적

일반 소비자가 **상품을 탐색하고 구매**하는 화면에 적용한다.
구매 전환과 사용자 경험이 핵심 목표다.

적용 대상:
- 소비자용 스토어 (Store Front)
- 상품 목록/상세 페이지
- 장바구니, 결제 플로우
- 랜딩 페이지, 홈페이지

---

## UI 판단 기준

### 정보 밀도
- **낮음~중간** - 핵심 정보만 노출
- 시각적 계층 명확
- 스크롤 기반 탐색

### 여백
- **넉넉함** - 시각적 여유, 호흡감
- 컴포넌트 간 간격: 24-32px
- 섹션 간 간격: 48-64px

### 색상 사용
- **브랜드 + 감성** - 브랜드 아이덴티티 강조
- Hero 섹션, CTA 버튼에 강한 색상
- 이미지, 사진 적극 활용
- 배경에 그라데이션, 색상 블록 허용

### 톤
- **친근함, 매력적**
- 마케팅 문구, 감성적 카피 허용
- 사용자 행동 유도 (CTA) 명확
