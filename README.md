# O4O Platform

## 프로젝트 개요
O4O(Online for Offline) 플랫폼 프로젝트입니다.

## 기술 스택
- Frontend: React.js
- Backend: Node.js
- Database: MongoDB

## 프로젝트 구조
```
o4o-platform/
├── frontend/        # React 프론트엔드
└── backend/         # Node.js 백엔드
```

## 시작하기

### 프론트엔드 실행
```bash
cd frontend
npm install
npm start
```

### 백엔드 실행
```bash
cd backend
npm install
npm start
```
