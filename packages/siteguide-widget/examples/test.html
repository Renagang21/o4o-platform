SiteGuide란?

SiteGuide는 방문자가 사이트를 이해하고 원하는 정보를 찾을 수 있도록 돕는 AI 안내 도우미입니다.

오른쪽 하단의 버튼을 클릭하여 AI 안내 위젯을 열어보세요!

주요 기능

  • 페이지 맥락 기반 자연어 질의응답
  • 자동 페이지 메타데이터 수집
  • 다크/라이트 테마 지원
  • 모바일 반응형 디자인
  • 쉬운 설치 (스크립트 1줄)

테스트 질문 예시

다음과 같은 질문을 해보세요:

  • "이 사이트는 뭐하는 곳이야?"
  • "SiteGuide의 주요 기능이 뭐야?"
  • "어떻게 설치하면 돼?"
  • "가격은 어떻게 돼?"
