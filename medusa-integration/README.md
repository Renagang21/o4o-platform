# Medusa Integration

Medusa 커머스 엔진을 기반으로 한 멀티벤더/멀티스토어 확장 모듈입니다.

## 구조

### 확장 모듈
- `extensions/`: Medusa 코어 확장
  - 멀티벤더 기능
  - 멀티스토어 기능
  - 어필리에이트 기능
  - 수수료 관리 기능

### 커스터마이징
- `customizations/`: 커스텀 기능
  - 커스텀 엔티티
  - 커스텀 API
  - 커스텀 서비스
  - 이벤트 핸들러

### 설정
- `config/`: 설정 파일
  - Medusa 설정
  - 데이터베이스 설정
  - 스토리지 설정
  - 결제 설정

## 주요 기능
- 멀티벤더 지원
  - 판매자별 상품/주문 관리
  - 판매자별 정산 관리
  - 판매자별 설정 관리

- 멀티스토어 지원
  - 스토어별 도메인 관리
  - 스토어별 테마 관리
  - 스토어별 상품 관리
  - 스토어별 주문 관리

- 어필리에이트 프로그램
  - 프로그램 관리
  - 커미션 관리
  - 실적 추적

## 기술 스택
- Medusa.js
- PostgreSQL
- Redis
- Node.js
- TypeScript
