# Marketplace Frontend

멀티벤더 및 멀티스토어 기반의 마켓플레이스 프론트엔드 모듈입니다.

## 구조

### 판매자 관련
- `vendor-portal/`: 판매자 대시보드 및 관리 인터페이스
  - 상품 관리
  - 주문 관리
  - 재고 관리
  - 정산 관리

### 스토어 관련
- `store-management/`: 스토어 관리 인터페이스
  - 스토어 설정
  - 디자인 커스터마이징
  - 상품 진열 관리
  - 프로모션 관리

### 구매자 관련
- `customer/`: 구매자 인터페이스
  - 상품 검색 및 조회
  - 장바구니
  - 주문 및 결제
  - 마이페이지

### 어필리에이트 관련
- `affiliate-portal/`: 어필리에이트 파트너 대시보드
  - 프로그램 등록/관리
  - 실적 조회
  - 정산 관리
  - 마케팅 도구

### 관리자 관련
- `admin/`: 플랫폼 관리자 인터페이스
  - 판매자 관리
  - 스토어 관리
  - 주문/결제 모니터링
  - 정산 관리
  - 시스템 설정

## 기술 스택
- React/Next.js
- TypeScript
- Tailwind CSS
- Redux/Redux Toolkit
- React Query
