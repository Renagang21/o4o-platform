# Digital Signage Frontend

디지털 사이니지 서비스의 프론트엔드 모듈입니다.

## 구조

### 관리자 페이지
- `admin/`: 관리자 인터페이스
  - 콘텐츠 관리
  - 디바이스 관리
  - 스케줄링
  - 모니터링
  - 통계/리포트

### 디스플레이 페이지
- `display/`: 디스플레이 인터페이스
  - 콘텐츠 렌더링
  - 실시간 업데이트
  - 오프라인 캐싱
  - 상태 보고

## 주요 기능
- 콘텐츠 관리
  - 미디어 업로드/관리
  - 템플릿 편집
  - 플레이리스트 관리
  - 콘텐츠 스케줄링

- 디바이스 관리
  - 디바이스 등록/설정
  - 상태 모니터링
  - 원격 제어
  - 그룹 관리

- 모니터링/분석
  - 실시간 상태 확인
  - 재생 통계
  - 성능 분석
  - 문제 진단

## 기술 스택
- React/Next.js
- TypeScript
- Tailwind CSS
- WebGL/Three.js
- WebSocket
- IndexedDB
- PWA
