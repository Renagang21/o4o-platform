# Community Frontend

커뮤니티 기능을 담당하는 프론트엔드 애플리케이션입니다.

## 주요 기능
- 포럼 및 게시판
- 댓글 및 답글
- 좋아요 및 북마크
- 사용자 프로필
- 알림 시스템

## 컴포넌트 구조
- `components/`: 재사용 가능한 UI 컴포넌트
- `pages/`: 라우트 기반 페이지 컴포넌트
- `hooks/`: 커스텀 훅
- `services/`: API 통신 서비스
- `utils/`: 유틸리티 함수

## 의존성
- `common/components`: 공통 UI 컴포넌트
- `common/hooks`: 공통 커스텀 훅
- `common/utils`: 공통 유틸리티 함수
