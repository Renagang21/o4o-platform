# Crowdfunding Frontend

크라우드펀딩 서비스의 프론트엔드 모듈입니다.

## 구조

### 관리자 페이지
- `admin/`: 관리자 인터페이스
  - 프로젝트 심사/승인
  - 창작자 관리
  - 결제/정산 관리
  - 통계/리포트
  - 시스템 설정

### 사용자 페이지
- `user/`: 사용자 인터페이스
  - 프로젝트 탐색/검색
  - 프로젝트 생성/관리 (창작자)
  - 후원하기/후원 관리 (후원자)
  - 커뮤니티/업데이트
  - 결제/정산

## 주요 기능
- 프로젝트 관리
  - 프로젝트 에디터
  - 리워드 설정
  - 프로젝트 대시보드
  - 업데이트 관리

- 후원 시스템
  - 후원하기
  - 리워드 선택
  - 결제 처리
  - 후원 현황

- 커뮤니티
  - 댓글/피드백
  - 업데이트 알림
  - 창작자-후원자 소통
  - 공지사항

## 기술 스택
- React/Next.js
- TypeScript
- Tailwind CSS
- Redux/Redux Toolkit
- React Query
- Socket.io (실시간 기능)
