# Marketplace Backend

멀티벤더 및 멀티스토어 기반의 마켓플레이스 백엔드 모듈입니다.

## 구조

### 판매자 관리
- `vendors/`: 판매자 관련 기능
  - 판매자 등록/인증
  - 판매자 프로필 관리
  - 판매자 권한 관리
  - 판매자 평가/리뷰

### 스토어 관리
- `stores/`: 스토어 관련 기능
  - 스토어 생성/설정
  - 스토어 테마 관리
  - 도메인 관리
  - 통계/분석

### 상품 관리
- `products/`: 상품 관련 기능
  - 상품 CRUD
  - 카테고리 관리
  - 재고 관리
  - 가격/할인 관리

### 주문 관리
- `orders/`: 주문 관련 기능
  - 주문 처리
  - 배송 관리
  - 반품/교환
  - 주문 상태 추적

### 결제 관리
- `payments/`: 결제 관련 기능
  - 결제 처리
  - 환불 처리
  - PG사 연동
  - 정산 처리

### 어필리에이트
- `affiliate/`: 어필리에이트 관련 기능
  - 프로그램 관리
  - 커미션 설정
  - 실적 추적
  - 파트너 관리

### 수수료 관리
- `commission/`: 수수료 정책 관리
  - 수수료 계산
  - 정산 주기 관리
  - 수수료 정책 설정

### 분석
- `analytics/`: 데이터 분석 기능
  - 매출 분석
  - 트래픽 분석
  - 상품 분석
  - 고객 행동 분석

## 기술 스택
- Node.js/Express
- TypeScript
- PostgreSQL
- Redis
- ElasticSearch
- RabbitMQ/Redis Pub/Sub
